# 牧场物语

《牧场物语》系列的首作。
舞台是 “花之芽村”，讲述了一个以成为牧场主为目标的男孩，用 3 年时间重建祖父留下的牧场的故事。

（日）牧場物語
（美）Harvest Moon

平台：

- [Super Famicom](https://www.bokumono.com/sub/bokumono.php) 1996.8.6
- Super Nintendo Entertainment System 1997.6.17
- [Nintendo Switch Online](https://www.nintendo.com/jp/games/feature/nintendo-classics/s-2037_j/index.html) 2022.3.31
